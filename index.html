How to use:

1. Click an element button, name it, and it appears on the canvas.

2. Drag elements to move them; double-click to rename inline.

3. Hover an element and drag from a port (small dot) to another element to connect them.

4. Right-click an element for quick actions; the right panel has the full editor.

5. Ctrl+Z / Ctrl+Y undo and redo. Delete removes, Escape deselects.
